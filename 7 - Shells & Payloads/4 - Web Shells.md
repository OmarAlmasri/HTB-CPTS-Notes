# Laudanum

Collection of ready-made files that can be inject into victim and receive access back via webshell, contains many different languages `aspx, asp, jsp, php,...`
Can be found in `/usr/share/laudanum`.

# Antak

We can use the  `Nishang Antak aspx` web shell for more advanced features and ability to execute commands in PowerShell-like environment.

# PHP Web Shells

We can use [WhiteWinterWolf's PHP Web Shell](https://github.com/WhiteWinterWolf/wwwolf-php-webshell)
